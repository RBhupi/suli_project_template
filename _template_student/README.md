# Student Research Folder

This folder contains all your research work for the internship. Keep it organized — your mentor reviews it weekly through pull requests.

## Folder Contents

| File / Folder | Purpose |
|---|---|
| `blog.md` | Weekly research log — update this every week before submitting your PR |
| `notes.md` | Personal scratchpad — references, commands, ideas, meeting notes |
| `pictures/` | Instrument photos, plots, screenshots for your poster or report |
| `deliverables/` | Final outputs — upload your poster, report, and abstract here before you leave |

## Weekly Workflow

1. Pull the latest changes from `main`
2. Work on your branch (`yourname/week03`)
3. Update `blog.md` with what you did this week
4. Add any pictures or notes
5. Push and open a pull request — link it to your weekly issue (`Closes #N`)

See [CONTRIBUTING.md](../CONTRIBUTING.md) for branch naming and commit style.
